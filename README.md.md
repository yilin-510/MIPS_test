# 项目介绍
+ 本项目主要参考雷思磊的《自己动手写CPU》一书，设计修改完成书中的教学版的针对MIPS指令集的CPU设计。
+ 相关实现功能以及设计仿真测试请见[系统硬件综合设计报告](./系统硬件综合设计报告.pdf)
+ 修改完善了书中的教学版OpenMIPS内容，对比书中内容，本项目在多处进行了修改，确保能够完美完成书中教学所想要的达到的效果，以及综合，仿真成功，并且能够成功生成二进制文件，烧录到EGO1板子上，显示指令运行效果
# 快速开始
+ 将测试的机器码.data文件放置.\project_OpenMIPS.srcs\sources_1\new目录下，请将inst_rom中`$readmemh("inst_rom.data", inst_mem);`修改为添加的测试机器码文件名
+ 即可开始综合仿真等工作
